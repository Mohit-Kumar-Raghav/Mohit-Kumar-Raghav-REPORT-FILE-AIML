# Climate Analysis and Prediction using Machine Learning

## Description
This project analyzes historical climate data and predicts future temperature using Machine Learning. Data preprocessing, feature selection, and Random Forest Regression are used to build the model. Visualizations are created to show trends and prediction accuracy.

## Features
- Climate data preprocessing  
- Temperature prediction using ML  
- Random Forest Regression model  
- Data visualization  
- Model evaluation  

## Technologies Used
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn  
- Scikit-learn  

## Project Structure
```
├── climate_analysis.py
├── dataset.csv
├── report.pdf
└── README.md
```

## How to Run
1. Install required libraries
```
pip install numpy pandas matplotlib seaborn scikit-learn
```

2. Run the file
```
python climate_analysis.py
```

## Output
- Temperature prediction
- Graph visualization
- Model accuracy results
